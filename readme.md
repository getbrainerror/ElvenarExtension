# About The Project

This is a chrome bot extension that lets you play Elvenar while skipping the most tedious tasks of the game. 

Our bot can help all of your neighbours in a few seconds with a single button click. 

And best of all, you get to enjoy the in-game bonuses without the need for actually watching their ads. Let the bot watch them for you!

# Installation

1. Download this repo (click the green buton that reads "code" -> then click "Download zip")
2. Extract the zip file you just downloaded in a folder (this folder should not change location, otherwise the extension won't find the files)
3. go to your chrome browser, click on the extension icon next to the search bar and  click on "manage extensions"
4. On this extension manager page, toggle "Developer mode" in the top-right corner
5. Click on "load unpacked"
6. select your unzipped extension folder


<!-- USAGE EXAMPLES -->
# Usage
Open the extension on an Elvenar game page and you'll see two sections:

### Ad watcher

The first section is where the bot can watch ad for you. There are 3 options: 
- knowledge points (5 ads/day)
- builders bonus on some productions (6 ads/day)
- city treasures (popping randomly or regularly)

For each of those cards, you can select the amount of ads to watch. For the knowledge points and builder bonus, you can also select what the bonus applies to

Once your selection is done, click the green play button. The ads are watched automatically in the background.

If you want to see the changes displayed in-game, you might need to reload the page

### Neighbourly help:

At the bottom of the extension popup, you'll see a handshake icon. Click it, and the extension will help all available neighbours. This should take under a minute.

You can choose to wait until the end of the loading screen to see if you got some treasures and which ones. Otherwise, you can also close the popup and the neighbourly help will keep processing in the background.

Again, you might need to reload the page if you want to see your resources correctly displayed by the game
